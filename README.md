# Seed-VC Pinokio App

This is a 1-click Pinokio app installer for [Seed-VC](https://github.com/Plachtaa/seed-vc), an AI voice conversion technology. 
It supports zero-shot voice conversion, real-time voice conversion, and singing voice conversion seamlessly on your local machine.

![Seed-VC App Icon](icon.png)

## Features
- **1-Click Installation:** Automatically clones the source repository and sets up the required virtual environment.
- **Integrated Web UI:** Easily access both v1 and v2 models through the bundled web interface.
- **GPU Optimization:** Automatically installs the optimal PyTorch backend for your system (supports Windows/Linux with Triton acceleration).

## Installation
You can install this app easily from the [Pinokio](https://pinokio.computer/) ecosystem.
